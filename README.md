## 🌟 About

Profile Craft is a modern drag & drop GitHub README.md builder that helps developers create beautiful GitHub profile pages easily.

With Profile Craft, you can design your profile visually, customize sections, and export a ready-to-use README.md file.

---

## 🎯 Why Profile Craft?

Creating a professional GitHub profile can be challenging, especially for beginners.

Profile Craft makes it easier:

✅ No Markdown skills required  
✅ Visual editor experience  
✅ Fast README generation  
✅ Modern designs  
✅ Customizable components  

---

## 🚀 Features

### 🖱️ Drag & Drop Editor

Create your README visually:

- Add sections
- Move components
- Customize layouts
- Edit content instantly

### 🧩 Components

Available blocks:

- 👤 About Me
- 💻 Skills
- 🚀 Projects
- 📊 GitHub Stats
- 🏆 Achievements
- 🌐 Social Links
- 📬 Contact Section
- ⚡ Custom Markdown

### 🎨 Customization

Personalize your README:

- 🌙 Dark Mode
- ☀️ Light Mode
- 🎨 Multiple Themes
- 🌍 Multi-language Support
- 🔤 Custom Styles
